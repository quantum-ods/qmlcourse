–ó–∞–ø–∏—à–µ–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–≥–æ –æ–∂–∏–¥–∞–Ω–∏—è —á–µ—Ä–µ–∑ —Å–æ—Å—Ç–æ—è–Ω–∏–µ \(\Psi_i\), –∫–æ—Ç–æ—Ä–æ–µ –ø—Ä–∏—à–ª–æ –Ω–∞ –≤—Ö–æ–¥ \(i\)-–≥–æ –≥–µ–π—Ç–∞ –≤ –Ω–∞—à–µ–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏:

\[ \langle M(\theta) \rangle = Tr(M U_{k, ..., 1} \rho_i U_{k, ..., 1}^\dagger) \]

–¢–æ–≥–¥–∞ —á–∞—Å—Ç–Ω–∞—è –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è –æ—Ç –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–≥–æ –æ–∂–∏–¥–∞–Ω–∏—è –ø–æ \(i\)-–º—É –ø–∞—Ä–∞–º–µ—Ç—Ä—É \(\theta_i\) –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è (–ø–æ–¥—Ä–æ–±–Ω–µ–µ –≤ [MNKF18]) —á–µ—Ä–µ–∑ –∫–æ–º–º—É—Ç–∞—Ç–æ—Ä –∏—Å—Ö–æ–¥–Ω–æ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏—è \(\rho\), –∫–æ—Ç–æ—Ä–æ–µ ‚Äú–ø—Ä–∏—à–ª–æ‚Äù –Ω–∞ –≤—Ö–æ–¥ –≥–µ–π—Ç–∞ \(U_i\) –∏ —Ç–æ–≥–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –ü–∞—É–ª–∏ \(P_i\), –∫–æ—Ç–æ—Ä—ã–π –º—ã –∏—Å–ø–æ–ª—å–∑—É–µ–º –≤ \(U_i\):

\[ \frac{\partial \langle M \rangle}{\partial \theta_i} = -\frac{i}{2}Tr(M U_{k, ..., i}[P_i, U_{i-1, ..., 1}\rho_i U_{i-1, ..., 1}^\dagger]U_{k, ..., i}^\dagger) \]

–≠—Ç–æ—Ç –∫–æ–º–º—É—Ç–∞—Ç–æ—Ä –º–æ–∂–µ—Ç –±—ã—Ç—å –ø–µ—Ä–µ–ø–∏—Å–∞–Ω —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

\[ [P_i, \rho] = i[U_i \Bigl( \frac{\pi}{2} \Bigr ) \rho_i U_i^\dagger \Bigl ( \frac{\pi}{2} \Bigr ) - U_i \Bigl( -\frac{\pi}{2} \Bigr ) \rho_i U_i^\dagger \Bigl ( -\frac{\pi}{2} \Bigr )] \]

–¢–æ–≥–¥–∞ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–π –≥—Ä–∞–¥–∏–µ–Ω—Ç \(\frac{\partial}{\partial \theta_i}\) –º–æ–∂–Ω–æ –∑–∞–ø–∏—Å–∞—Ç—å —á–µ—Ä–µ–∑ —Å–º–µ—â–µ–Ω–∏—è –Ω–∞ \(\pm\frac{\pi}{2}\):

\[\begin{split} \begin{gathered} \frac{\partial \langle M \rangle}{\partial \theta_i} = \frac{\langle M_i^+ \rangle - \langle M_i^- \rangle}{2} \\ \langle M_i^{\pm} \rangle = \frac{1}{2} Tr [M U_{k, ..., i+1} U_i(\pm \frac{\pi}{2})\rho_i^` U_i^\dagger (\pm \frac{\pi}{2}) U_{k, ..., i+1}^\dagger] \\ \rho_i^` = U_{j, ..., 1} \rho_i U_{j, ..., 1}^\dagger \end{gathered} \end{split}\]

–ü–æ –∞–Ω–∞–ª–æ–≥–∏–∏ —Å –∫–ª–∞—Å—Å–∏—á–µ—Å–∫–∏–º–∏ –Ω–µ–π—Ä–æ–Ω–Ω—ã–º–∏ —Å–µ—Ç—è–º–∏ –∏ backpropagation (–¥–ª—è —Ç–µ—Ö, –∫—Ç–æ –∑–∞–±—ã–ª —ç—Ç–æ –ø–æ–Ω—è—Ç–∏–µ, —Ä–µ–∫–æ–º–µ–Ω–¥—É–µ–º –≤–µ—Ä–Ω—É—Ç—å—Å—è –∫ –≤–≤–æ–¥–Ω—ã–º –ª–µ–∫—Ü–∏—è–º–∏ –ø—Ä–æ –∫–ª–∞—Å—Å–∏—á–µ—Å–∫–æ–µ –º–∞—à–∏–Ω–Ω–æ–µ –æ–±—É—á–µ–Ω–∏–µ) —Ç—É—Ç —è–≤–Ω–æ –º–æ–∂–Ω–æ –≤—ã–¥–µ–ª–∏—Ç—å forward –ø—Ä–æ—Ö–æ–¥ —Å–æ —Å–º–µ—â–µ–Ω–∏–µ–º \(\theta_i\) –Ω–∞ –∑–Ω–∞—á–µ–Ω–∏—è \(\frac{\pi}{2}\) –∏ backward —Å–æ —Å–º–µ—â–µ–Ω–∏–µ–º –Ω–∞ \(-\frac{\pi}{2}\).

–û–±–æ–±—â–µ–Ω–Ω—ã–π parameter-shift¬∂

–ü—Ä–µ–¥–ª–æ–∂–µ–Ω–Ω–æ–µ –≤ [MNKF18] –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–∞ —Å–∞–º–æ–º –¥–µ–ª–µ –ø–æ–ª—É—á–µ–Ω–æ –≤ –±–æ–ª–µ–µ –æ–±—â–µ–º –≤–∏–¥–µ –∏–∑ –¥—Ä—É–≥–∏—Ö —Å–æ–æ–±—Ä–∞–∂–µ–Ω–∏–π. –¢–∞–∫, –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –¥–ª—è –Ω–∞—à–µ–π –Ω–∞–±–ª—é–¥–∞–µ–º–æ–π \(\langle M \rangle\) –º–æ–∂–µ—Ç –≤—Å–µ–≥–¥–∞ –±—ã—Ç—å –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–æ [MBK21] –∫–∞–∫ —Å—É–º–º–∞ –≤–∏–¥–∞:

\[ \bra{U_i(\theta_i)}\hat{M}\ket{U_i(\theta_i)} = \hat{A} + \hat{B}\cos{\theta_i} + \hat{C}\sin{\theta_i} \]

–≥–¥–µ \(\hat{A}, \hat{B}, \hat{C}\) ‚Äì –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã, –Ω–µ –∑–∞–≤–∏—Å—è—â–∏–µ –æ—Ç –ø–∞—Ä–∞–º–µ—Ç—Ä–∞ \(\theta_i\).

Note

–î–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ, —è–≤–Ω–æ –≤—ã–ø–∏—Å–∞–≤ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –¥–ª—è –Ω–∞–±–ª—é–¥–∞–µ–º–æ–π –∏ –≤—Å–ø–æ–º–Ω–∏–≤ —Ñ–æ—Ä–º—É–ª—ã –¥–ª—è –∫–æ—Å–∏–Ω—É—Å–∞ –∏ —Å–∏–Ω—É—Å–∞ –¥–≤–æ–π–Ω–æ–≥–æ —É–≥–ª–∞, –∞ —Ç–∞–∫–∂–µ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞–≤—à–∏—Å—å —Ç–µ–º, —á—Ç–æ \(U(\theta) = e^{-\frac{1}{2}H\theta} = \cos{\frac{\theta}{2}}\mathbf{1} - i\sin{\frac{\theta}{2}}H\), –ø–æ–ª—É—á–∞–µ–º:

\[\begin{split} \begin{gathered} (\cos{\frac{\theta}{2}}\mathbf{1} + i\sin{\frac{\theta}{2}}H)\hat{M}(\cos{\frac{\theta}{2}}\mathbf{1} - i\sin{\frac{\theta}{2}}H) = \\ \cos^2{\frac{\theta}{2}}\hat{M} + i \sin{\frac{\theta}{2}}\cos{\frac{\theta}{2}}H\hat{M} - i \sin {\frac{\theta}{2}}\cos{\frac{\theta}{2}}\hat{M}H + \sin^2{\frac{\theta}{2}}H\hat{M}H = \\ \frac{1}{2} \cos{\theta}\hat{M}+\frac{1}{2}\hat{M}+\frac{i}{2} \sin{\theta}H\hat{M} - \frac{i}{2}\sin{\theta}\hat{M}H + \frac{1}{2}H\hat{M}H - \frac{1}{2}\cos{\theta}H\hat{M}H = \\ \frac{1}{2}(\hat{M} + H\hat{M}H) + \frac{1}{2}(\hat{M} - H\hat{M}H)\cos{\theta} + \frac{i}{2}(H\hat{M} - \hat{M}H)\sin{\theta} \end{gathered} \end{split}\]

–¢–æ–≥–¥–∞ –º–æ–∂–Ω–æ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –ø—Ä–∞–≤–∏–ª–∞–º–∏ —Ç—Ä–∏–≥–æ–Ω–æ–º–µ—Ç—Ä–∏–∏, –∞ –∏–º–µ–Ω–Ω–æ, —Ç–µ–º —á—Ç–æ –¥–ª—è –ª—é–±–æ–≥–æ \(s \neq k\pi, \text{ } k \in {1, 2, ..., }\) —Å–ø—Ä–∞–≤–µ–¥–ª–∏–≤–æ:

\[\begin{split} \begin{gathered} \frac{d \cos {\theta}}{d \theta} = \frac{\cos (\theta + s) - \cos (\theta - s)}{2\sin{s}} \\ \frac{d \sin {\theta}}{d \theta} = \frac{\sin (\theta + s) - \sin (\theta - s)}{2\sin{s}} \\ \end{gathered} \end{split}\]

\[ \frac{\partial \langle M(\theta_i) \rangle}{\partial \theta_i} = \frac{\langle M(\theta_i + s) \rangle - \langle M(\theta_i - s) \rangle}{2\sin{s}} \]
